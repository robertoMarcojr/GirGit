Keyboard Shortcuts
  • Ctrl + ↑ — Move window up
  • Ctrl + ↓ — Move window down
  • Ctrl + ← — Move window left
  • Ctrl + → — Move window right
  • Ctrl + Shift + ↑ — Scroll up
  • Ctrl + Shift + ↓ — Scroll down
  • Ctrl + / — Toggle main window visibility (hide/show)
  • Ctrl + Q — Quit app
  • Ctrl + D — Delete all screenshots (reset queue)
  • Ctrl + Backspace — Delete last screenshot
  • Ctrl + Shift + M — Toggle mouse blocking
  • Ctrl + L — Take screenshot
  • Ctrl + Enter — Send prompt
  • Ctrl + Space — Toggle audio recording (start/stop)

ℹ️ All windows are resizable and movable.

⌨️ This app is designed to be primarily keyboard-driven. Press Ctrl + Shift + M to make it interactable.

⚠️ Enabling interaction may cause the app to appear in the taskbar. For a stealth experience, keep it keyboard-driven.
